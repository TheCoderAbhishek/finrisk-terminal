# FinRisk Terminal — Product Specification

## Overview

FinRisk Terminal is a frontend-only Angular intelligence platform that combines public company financial data, SEC filing activity, macroeconomic indicators, and climate-risk signals into a modern analyst-grade dashboard.

## Objective

The objective is to build a scalable, accessible, high-performance frontend application using Angular, RxJS, TypeScript, Axios, Tailwind CSS, Angular Signals, and public APIs only.

## Target Users

- Investors
- Financial analysts
- ESG researchers
- Students
- Enterprise decision-makers
- Technical recruiters and interviewers

## Core Problems Solved

- Public company data is fragmented across different sources.
- SEC data is difficult to interpret for non-experts.
- Financial performance is rarely shown together with macroeconomic and climate-risk context.
- Many portfolio projects are CRUD-focused and do not demonstrate real-world frontend engineering.

## MVP Features

1. Market Intelligence Dashboard
2. Company Search
3. Company Profile
4. SEC Filing Timeline
5. Financial Snapshot
6. Macro Indicator Explorer
7. Climate-Risk Overview
8. Local Watchlist
9. Mock Session Handling
10. Responsive Premium UI

## Out of Scope for MVP

- Real authentication
- Backend APIs
- Database integration
- Paid APIs
- Investment advice
- Real trading functionality
- User account management

## Disclaimer

This application is built for educational and portfolio purposes only. It does not provide investment, financial, legal, or professional advice.
