# FinRisk Terminal — UI/UX Guidelines

## Visual Direction

FinRisk Terminal should look like a premium financial analytics platform.

## Style Keywords

- Modern
- Premium
- Dark-first
- Glassmorphism
- Subtle gradients
- Dense but readable
- Analyst-grade
- Accessible
- Responsive
- Motion-rich but not distracting

## Core UI Elements

- Dashboard cards
- KPI widgets
- Risk chips
- Filing timeline
- Financial charts
- Comparison panels
- Search command palette
- Skeleton loaders
- Toast notifications
- Empty states
- Error states

## Accessibility Rules

- All interactive elements must be keyboard accessible.
- Focus states must be visible.
- Charts must include text summaries.
- Color must not be the only way to communicate risk.
- Reduced-motion preference must be respected.
- Tables must use proper headers and captions.
